CI/CD

CI/CD (Continuous Integration / Continuous Delivery) — это практика автоматизации процессов сборки, тестирования и деплоя кода.

Основные команды

  • git push – Запуск CI/CD pipeline после отправки кода
  • ci start – Запустить процесс CI (пример, зависит от платформы)
  • ci status – Проверить статус pipeline
  • ci stop – Остановить процесс CI
  • ci logs – Просмотреть логи выполнения pipeline

Популярные команды

  • ci test – Запустить автотесты
  • ci deploy – Запустить деплой
  • ci artifacts – Скачать артефакты сборки
