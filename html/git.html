Git

Работа с репозиторием

  • git init – Инициализировать новый репозиторий
  • git clone – Клонировать репозиторий
  • git status – Статус файлов в репозитории
  • git add – Добавить файл в индекс
  • git commit – Зафиксировать изменения
  • git push – Отправить изменения на сервер
  • git pull – Получить изменения с сервера
  • git branch – Управление ветками
  • git checkout – Переключиться на ветку
  • git merge – Объединить ветки

История и сравнение

  • git log – История коммитов
  • git diff – Показать различия
  • git tag – Управление тегами

Работа с удалёнными репозиториями

  • git remote – Управление удалёнными репозиториями
  • git fetch – Получить объекты из другого репозитория
  • git revert – Откатить коммит
  • git reset – Сбросить индекс
  • git stash – Временное сохранение изменений
